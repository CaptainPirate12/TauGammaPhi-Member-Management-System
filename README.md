# TauGammaPhi Member Management System

## Project Overview

The TauGammaPhi Member Management System is a comprehensive web-based platform designed to streamline member registration, profile management, and certificate generation for the Tau Gamma Phi fraternity.

## Features

- User Registration & Authentication
- Member Dashboard
- Profile Management
- Certificate Generation
- Responsive Design

## Technology Stack

- Frontend: HTML5, CSS3, JavaScript
- Backend: PHP
- Database: MySQL
